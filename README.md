# Metasploit

Objetivo

El objetivo principal de Metasploit es facilitar las pruebas de penetración y la auditoría de seguridad mediante la automatización de la explotación de vulnerabilidades.
A diferencia de otras herramientas que solo encuentran fallos (como un escáner de vulnerabilidades), Metasploit permite validar si esas vulnerabilidades son reales y qué impacto tendrían si fueran explotadas por un atacante.

### Habilidades Aprendidas

- Montaje y configuración de laboratorios virtuales.
- Pruebas de conectividad y diagnóstico de red.
- Escaneo de puertos y enumeración de servicios con Nmap.
- Identificación y análisis de versiones de software vulnerables.
- Navegación y gestión del Framework Metasploit.
- Búsqueda y selección de módulos de explotación y auxiliares.
- Configuración de parámetros de ataque (RHOSTS, RPORT, LHOST).
- Explotación de servicios FTP y aplicaciones web (Tomcat).
- Gestión de sesiones de Command Shell y Meterpreter.
- Verificación y escalada de privilegios (root).
- Ejecución de conceptos de post-explotación.
- Técnicas de persistencia y limpieza de huellas.

### Herramientas Utilizadas

- Kali Linux
- Metasploitable 2
- Nmap
- Metasploit Framework (msfconsole)
- Meterpreter
- Ping
- ip a / ifconfig
- vsftpd
- Apache Tomcat
